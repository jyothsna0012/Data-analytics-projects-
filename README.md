**Tools:** Excel, Power BI

Analyzed sales data to identify revenue trends, profit margins, regional performance, and top-selling products through an interactive dashboard.

- Monthly Sales Trend

- Sales by Region
- Profit Analysis
- Interactive Filters

### 🛒 2. E-Commerce Conversion Rate Analysis
**Tools:** Excel, SQL, Power BI

**Description:**
Analyzed customer behavior across the purchase funnel to identify conversion bottlenecks and improve business performance.

**Key Features**
- Conversion Funnel
- Bounce Rate Analysis
- Device Performance

- Revenue Metrics

**Key Features**
- Hypothesis Testing
- Conversion Comparison
- Confidence Analysis
- Business Recommendations

**Tools:** Python, Scikit-learn

Built a machine learning model to predict customers likely to leave a business using historical customer data.

**Key Features**
- Data Cleaning

- Model Training

- Churn Prediction
