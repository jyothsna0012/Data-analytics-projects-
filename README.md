# 📊 Sales Trend Visualization Dashboard

A Data Analytics project that analyzes sales performance using Power BI. This dashboard provides interactive visualizations and key business insights to help understand sales trends, profitability, customer behavior, and regional performance.

---

## 📌 Project Overview

The goal of this project is to transform raw sales data into meaningful business insights through data cleaning, analysis, and interactive dashboards.

This dashboard helps answer questions such as:
- How much total sales were generated?
- Which products generate the highest profit?
- Which regions perform the best?
- How do sales change over time?
- Which product categories are most profitable?

---

## 🎯 Objectives

- Analyze sales performance over time.
- Track business KPIs.
- Identify top-performing products and categories.
- Compare sales across different regions.
- Visualize profit trends.
- Build an interactive dashboard for business decision-making.

## 📈 Key Insights

- Technology category generated the highest sales.
- West region recorded the highest revenue.
- Some products generated negative profit due to high discounts.
- Sales showed consistent growth during the year.
- Office Supplies contributed a large number of orders.
